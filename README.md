# M485-BLOC6-DJANGO-Projecte-Blog
## Introducció

Aquest projecte és una aplicació web desenvolupada amb Django que permet gestionar un blog amb publicacions, autors i etiquetes.

L'aplicació permet:

- Visualitzar les darreres publicacions
- Consultar totes les entrades del blog
- Visualitzar autors
- Filtrar publicacions per etiquetes
- Accedir als detalls de cada publicació

## Tecnologies utilitzades

- Python 3
- Django 6
- SQLite3
- HTML5
- CSS3
